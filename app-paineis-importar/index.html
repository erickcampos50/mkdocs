Como importar painéis de controle para o aplicativo¶

Neste tutorial vamos explicar como importar o conjunto de recursos visuais que irá permitir utilizar os dados criados pelos sensores, aqui denominaremos eles de painel ou dashboard, como os vistos na imagem abaixo. Esta é a próxima etapa após a configuração básica do aplicativo, a partir daqui que será possível acessar os dados coletados pelos sensores.

Desta forma você poderá acompanhar remotamente tudo o que acontece nos equipamentos que estão sendo monitorados de uma forma acessível e visual.

Para importação, é preciso ter acesso a um endereço digital de onde se encontram as configurações do painel que irá utilizar, com este endereço o aplicativo irá fazer o download de todas as informações diretamente do servidor. Essa funcionalidade permite também atualizar os painéis quando, por exemplo, houver algum sensor novo instalado. A atualização dos painéis será abordada posteriormente em Como atualizar os painéis.

Lista de endereços disponíveis

Escolha um dos endereços abaixo para inserir no aplicativo quando estiver executando o passo a passo:

Painel principal do RU-GV: app/dashboard/RU/painel1

Painel sucinto do RU-GV: app/dashboard/RU/painel2 (contém somente as informações mínimas visualizar os dados coletados, dispensando controles e gráficos)

Painel principal da sala do servidor de internet UFJF-GV: app/dashboard/UFJF/servidor1
