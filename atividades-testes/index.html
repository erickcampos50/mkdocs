Atividades que devem ser realizadas durante a etapa de testes experimentais¶

O propósito dos testes com os usuários é explorar os dispositivos e também sistemas num ambiente real, por isso é parte essencial do experimento que os usuários façam uso de todas os recursos para que possam contribuir com sua opinião após a finalização dos testes.

Apesar de serem simples, as orientações abaixo são essenciais:

Todos devem utilizar todos os recursos que forem disponibilizados

Cada funcionário terá acesso a algum tipo de recurso, ficando a cargo do gestor da unidade definir quais serão. Portanto, se um funcionário terá acesso ao aplicativo ou não é uma definição discricionária do gestor, mas uma vez que se tenha definido quem vai poder ter acesso ao que é preciso instruir os funcionários a utilizar as interfaces do sistema que lhes foram disponibilizadas e reportar críticas e sugestões sobre os dispositivos.

É preciso que todos tenham uma opinião sobre o que foi disponibilizado e pra essa opinião se formar é necessário uso.

Condições de uso do aplicativo

Todos que tiverem acesso ao aplicativo devem utilizá-lo pelo menos 3 vezes por semana.

O uso se resume a tentar executar uma ação simples como acessar as informações ou ajustar um controle.

Os sensores não devem ser removidos

Nesta etapa de testes os sensores não podem ser removidos para instrumentar outros equipamentos

Qualquer problema com os dispositivos deve ser relatado imediatamente

Considerando que são dispositivos em fase experimental, é esperado que hajam problemas de integração ou uso que possam culminar na necessidade de desligamento dos sensores. Não há problema algum nisso.

No entanto, é compulsório que esse tipo de situação seja relatada imediatamente para permitir acompanhar a intercorrência e providenciar a solução.

Valores de referência para os alertas

Os diversos alertas disponibilizados podem ser modificados de acordo com o equipamento que estiver sendo monitorado. Se for um equipamento aquecedor o alerta de mal funcionamento não será idêntico a um equipamento de refrigeração, por isso há a possibilidade de modificar os controles. No entanto, todos os valores devem ser definidos respeitando a legislação.

Avaliação da funcionalidade de alertas

Todos os usuários que possuam acesso ao grupo de Telegram devem avaliar os alertas e a forma como eles ocorrem. Caso nenhuma intercorrência aconteça naturalmente, faremos uma simulação para garantir uma experiência mínima do usuário com essa funcionalidade.
